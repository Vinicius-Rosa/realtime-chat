## How to run

First, clone the repo:
> git clone https://github.com/Vinicius-Rosa/realtime-chat.git

Then, navigate to the folder:
> cd realtime-chat

#### The Server:
Navigate to the server folder:
> cd server

Start the server:
> yarn start

#### The Client:
Navigate to the client folder:
> cd client

Start the client:
> yarn start
